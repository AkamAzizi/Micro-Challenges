# Feel Good Kickstart

A simple app designed to help users build micro-habits and feel better every day.

## Features
- Track micro-habits
- Clean and simple UI
- Focused on consistency over complexity

## Tech Stack
- React Native (Expo)
- TypeScript / JavaScript

## Getting Started
1. Clone the repo  
2. Install dependencies: 
pm install  
3. Start the app: 
px expo start
